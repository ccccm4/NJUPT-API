3. 图书馆接口¶

class njupt.library.LoginType¶

图书馆的登录方式

  • STUDENT_ID: 学号

  • CARD_ID: 一卡通号

  • EMAIL: 邮箱

class njupt.Library(username, password, login_type=<LoginType.CARD_ID: 'bar_no'>)¶

图书馆

参数

  • username – 用户名

  • password – 密码

  • login_type – LoginType, 默认为一卡通账号
